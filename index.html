zip.js

A JavaScript library to zip and unzip files in the browser

Notable features

  • Support of the Zip64 format
  • Support of WinZIP AES-256 encryption
  • Worker pool manager
  • Parallel reads/writes in one or multiple zip files

This library depends on the Promise, the Typed array APIs and these APIs optionally:

  • Web workers
  • Web Crypto

Compatibility

This library works fully with the latest versions of Chrome, Firefox, Safari, and Microsoft Edge.

Demos

  • Read a zip file (Core API)
  • Zip Manager (Filesystem API)

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/gildas-lormeau/zip.js

You can also get the code via NPM by running:

$ npm install @zip.js/zip.js

zip.js

A JavaScript library to zip and unzip files

zip.js provides a low-level API based on promises for writing and reading large zip files efficiently.
zip-fs.js provides a high-level filesystem API.

Notable features

  • Support of the Zip64 format
  • Support of WinZIP AES-256 encryption
  • Web worker pool manager
  • Parallel writes in a zip file

This library depends on the Promise, the Typed array APIs and these APIs optionally:

  • Web workers
  • File
  • Crypto

Compatibility

This library works fully with the latest versions of Chrome, Firefox, Safari, and Microsoft Edge.

Install

  • add zip.min.js, z-worker.js, deflate.js, inflate.js, and crypto.js from the dist folder in your project
  • include zip.min.js script in your HTML page 
    <script type="text/javascript" src="/lib/zip.min.js"></script>
  • configure workerScriptsPath (i.e. the URL to the directory containing z-worker.js, deflate.js, inflate.js, and crypto.js) if needed. 
    zip.configure({
  workerScriptsPath: "/lib/"
});
